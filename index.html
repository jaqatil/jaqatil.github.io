Язык Уэ.тил / The Ue.til Language

Синтаксис и фонотактика 
      Текст = Предложение | ТекстПредложение
Предложение = Имя | ПредложениеОп1 | ПредложениеПредложениеОп2

        Имя = Слово кроме Оп1,Оп2
        Оп1 = ABC
        Оп2 = B | AC
      Слово = первая буква P, последняя Q, без QP
      
          P = A | B
          Q = B | C
          
          A = б ф х с т
          B =  а и у .
          C =   л н р
          X =    э о
Syntax and phonotactics 
       Text = Sentence | TextSentence
   Sentence = Noun | SentenceOp1 | SentenceSentenceOp2

       Noun = Word except Op1,Op2
        Op1 = ABC
        Op2 = B | AC
       Word = the first letter P, the last Q, without QP
      
          P = A | B
          Q = B | C
          
          A = b f h s t
          B =  a i u .
          C =   l n r
          X =    e o
